About Approval Workflows

Approval workflows define the approval process for RFPOs within each consortium. Each workflow consists of:

  • Stages - Budget bracket levels (e.g., "Under $5,000", "Up to $15,000")
  • Steps - Individual approval actions within each stage (e.g., "Technical Review", "Management Approval")
  • Approvers - Primary and backup users who can take approval actions

Only one workflow can be active per consortium at a time. When an RFPO is submitted, it uses the active workflow for its consortium.
